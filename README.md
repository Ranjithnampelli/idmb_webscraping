# IMDb Top Rated Movies Web Scraping Project

## Overview

This project is a web scraping application that fetches information about the top-rated movies from IMDb. IMDb (Internet Movie Database) is a popular online database of information related to films, television series, and video games. The purpose of this project is to retrieve data on the highest-rated movies from IMDb, including details such as title, rating, genre, and release year.

## Features

- **Web Scraping**: The project utilizes web scraping techniques to extract relevant information from the IMDb website.

- **Top Rated Movies**: The application specifically targets the list of top-rated movies, providing valuable insights into the best-rated films according to IMDb users.

- **Data Extraction**: Information such as movie title, rating, genre, and release year is extracted and organized for easy consumption.

- **Scalability**: The web scraping script is designed to be scalable, allowing for future updates and modifications to adapt to any changes in the IMDb website structure.

## How to Use

### Prerequisites

Make sure you have the following prerequisites installed:

- Python 3.x
- Beautiful Soup 4
- Requests library

### Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/your-username/imdb-top-rated-movies.git
   ```

2. Install the required dependencies:

   ```bash
   pip install beautifulsoup4 requests
   ```

### Usage

1. Navigate to the project directory:

   ```bash
   cd imdb-top-rated-movies
   ```

2. Run the script:

   ```bash
   python imdb_scraper.py
   ```

3. The script will fetch the top-rated movies from IMDb and display the extracted information.
